# Myntra Home Page Clone

A high-fidelity frontend clone of the Myntra homepage, built to practice responsive web design and modern CSS layouts.

## 🚀 Features
- **Myntra-Level Responsiveness**: A fully adaptive navigation bar that handles different screen sizes by prioritizing core categories and hiding secondary elements like the search bar on mobile.
- **Dynamic Layouts**: Uses Flexbox for product category grids and a comprehensive, responsive footer.
- **Interactive UI**: Includes hover states for navigation links and profile icons with signature branding colors.

## 🛠️ Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom styling, Flexbox, and Media Queries.
- **Google Fonts**: "Assistant" font family to match Myntra's branding.

## 📱 Responsive Logic
The project uses CSS Media Queries to ensure a seamless experience:
- **Desktop**: Full navigation with Search Bar and Action Icons.
- **Mobile (<768px)**: Streamlined header prioritizing brand and main categories with a touch-friendly horizontal scroll.

## 🔧 Installation
1. Clone the repository: `https://github.com/trinoy-kundu/myntra-clone-responsive`
2. Open `index.html` in your browser.
